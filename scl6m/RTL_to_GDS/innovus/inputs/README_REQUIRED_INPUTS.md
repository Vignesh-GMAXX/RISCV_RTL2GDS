# Required Inputs To Populate

Drop or link these files in this folder, OR set the corresponding env variables.

## Physical/timing collateral

- `tech.lef` (or set `LEF_TECH`)
- `stdcell.lef` (or set `LEF_CELL`)
- `max.lib` (or set `LIB_MAX`)
- `min.lib` (or set `LIB_MIN`)
- `qrcTechFile.tch` (or set `QRC_TECH_FILE`)
- `streamout.map` (or set `STREAMOUT_MAP`)

## Optional

- macro LEF(s) via `LEF_MACRO`
- GDS merge list in `config.tcl` (`GDS_MERGE_FILES`)
